# Weather Analytics Dashboard

An interactive weather dashboard built using Power BI and WeatherAPI.

## Features

- Current weather conditions
- 7-day forecast
- Air Quality Index monitoring
- PM10, PM2.5, CO, NO2, SO2, O3 metrics
- Sunrise and Sunset tracking
- Rain probability analysis
- Dynamic city selection

## Tools Used

- Power BI
- DAX
- Power Query
- WeatherAPI
